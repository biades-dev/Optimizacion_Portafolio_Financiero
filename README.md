# Optimizacion_Portafolio_Financiero
Proyecto de finanzas cuantitativas que construye la Frontera Eficiente. Se utiliza Python (Pandas, yfinance) para la extracción y procesamiento de datos históricos, y un modelo en Excel (Solver, MMULT) para calcular el portafolio de mínima varianza y optimizar la asignación de activos.
